# Fennex Browser
### Navegador web moderno escrito en Python y Qt6 (QtWebEngine)

Fennex Browser es un navegador multiplataforma completamente desarrollado en **Python**, usando **PyQt6** y **QtWebEngine (Chromium)**. 
Su objetivo es ofrecer una experiencia moderna, rápida y privada, con interfaz visual limpia, soporte para pestañas, historial, marcadores y gestión segura de contraseñas.

## Descripción

Fennex Browser es un navegador web open source hecho completamente en **Python**, 
usando **PyQt6 y QtWebEngine** (Chromium).  
Es uno de los navegadores más completos escritos en Python, ofreciendo interfaz moderna con pestañas, historial, marcadores, descargas y soporte para contraseñas cifradas.
